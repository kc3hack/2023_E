## プロダクト説明

<!-- プロダクトの説明を入力 -->
絵を描くときに構図に困ったことはありませんか？？  
この3D構図シミュレータは名前の通り3次元モデルによって構図の構想を練る際の手助けとなります。  
本Webアプリでは，マウスのドラッグ操作でカメラワークを自由に動かすことができ，人モデルにおいては，
構図、人体、背景を直感的に組み立てられるWebアプリです。
